# Termine (Archiv)

**Archiv aller bisherigen Therapieplan-Scans.** Hierhin wandert ein Scan aus
`../Update-Termine/`, nachdem Claude ihn in `plan.json` eingearbeitet hat.

Empfehlung: pro Ausdruck einen Unterordner mit Datum, z. B.
`2026-06-19_Ausdruck/`, damit die Historie nachvollziehbar bleibt.

**Wichtig:** Die Scans enthalten Privatdaten und sind per `.gitignore` vom
öffentlichen Repo ausgeschlossen. Nur diese README liegt im Repo.
